Principe

Toutes les compétences sont dans la FI ! Cet Incubateur a pour vocation de les centraliser et de les mettre en lien avec les projets militants qui en expriment le besoin.

Si vous vous sentez de lancer et gérer un projet, il vous suffit de renseigner les compétences qui vous manquent pour que les insoumis les possédant soient alertés.

La partie Idée est intermédiaire, proposez vos idées ici si vous n'avez pas les compétences ou le temps de gérer le projet vous-même

Les Outils

Nous mettons à disposition différents outils pour permettre à chacun de gérer ses actions militantes.

Cette liste, courte pour l'instant, a pour ambition de se développer et de répondre en temps réel aux besoins d'organisation des militants.
